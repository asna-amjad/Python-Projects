# Pong Game Project (Arcade Game)

### Concepts Practiced:

	•	Object-Oriented Programming (OOP) with multiple classes
	•	Class inheritance and reusable components
	•	turtle module for 2D game development
	•	Game loop and screen update control
	•	Collision detection (ball, paddles, walls)
	•	Keyboard event handling for player movement
	•	Score tracking and display updates
	•	Ball movement and speed control
	•	Code modularization across multiple files
