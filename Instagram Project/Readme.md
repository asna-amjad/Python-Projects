# Instagram Foller Bot

### Concepts Practiced
* Selenium WebDriver automation
* Browser interaction using Python
* Web scraping dynamic content
* Automate Filling Out Forms and Clicking Buttons with Selenium OOP
* Handling DOM elements and locators (XPath, CSS Selector, Tag Name)
* Handling pop-ups and alerts
* Exception handling in Selenium (ElementClickInterceptedException)
* Working with dynamically loaded content (AJAX)
